# MonHubEclipse

## Description
MonHubEclipse est une application web pour regrouper mes appli Python de photographie d'éclispe.

## Fonctionnalités
Accueil 😎

1. Génération des fichiers d'ordonnancement 🕔
2. Tests et configuration du filtre automatique 🛠️
3. Application de lecture des logs 📜

## Installation

1. Création d'un environnement virtuel

> python3 -m venv ~/hub_env

2. Activation le l'environnement

> source ~/hub_env/bin/activate
ou
> source ~/eclipse_env/bin/activate

3. les dépendances Python

> pip install streamlit
> pip install plotly.express

ou

> pip install -r requirements.txt

## Utilisation

> streamlit run Accueil.py
ou
> MHE_launcher.sh
