# Fichier de description de l'ordonnancement des actions
# Ce fichier est destiné à être lu par le script d'ordonnancement pour déclencher les actions aux heures spécifiées.
#
# Notes sur les paramètres :
# - Ouverture: f-number (ex: 4 = f/4, 5.6 = f/5.6)                      -> Vérifier avec un gphoto2 --get-config=aperture que les valeurs soient compatibles avec votre boitier
# - ISO: valeur ISO (ex: 1600, 800, 400)                                -> Vérifier avec un gphoto2 --get-config=iso que les valeurs soient compatibles avec votre boitier
# - Vitesse: en secondes (ex: 1 = 1s, 1/125s) (30->0.3-1/4->1/4000)     -> Vérifier avec un gphoto2 --get-config=shutterspeed que les valeurs soient compatibles avec votre boitier
# - MLU: délai Mirror Lock-Up en millisecondes (0 = désactivé)          -> Uniquement pour les boitiers avec miroir et compatible avec cette fonction
#
# Notes sur les actions
# Format: Type,reference_temps,operateur,temps,operateur_fin,temps_fin,intervalle_ou_nombre,ouverture,iso,vitesse,mlu
# Photo,C1,-,00:00:10,ouverture,iso,vitesse_obturation,delai_mlu
# Boucle,C1,-,00:00:10,+,00:00:00,Tps,ouverture,iso,vitesse_obturation,delai_mlu
# Interval,C2,-,00:01:00,+,00:01:00,Nb,ouverture,iso,vitesse_obturation,delai_mlu
# Filter,C2,-,00:00:10,action
#
